# Focus Flow (Anti-Procrastination Focus System)

A mobile-first React + TypeScript + Vite app that prioritizes deep work over task lists.

## Run

```bash
npm install
npm run dev
```

## Key Features

- Focus mode first with circular timer and immersive running state
- Exit guard with multi-step friction (confirm + hold / wait)
- Simulated app blocking overlay during active sessions
- Lightweight tasks with today/later grouping and priority
- Local analytics cards (sessions today, streak, tasks done today)
- Preloaded demo data stored in localStorage
